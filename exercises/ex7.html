Ex.7 : HTML : intermediate : สร้างการ์ดสินค้าด้วย Semantic

โจทย์นี้ท้าทายให้คุณสร้างการ์ดแสดงรายละเอียดสินค้าโดยใช้หลักการของ Semantic HTML เพื่อให้โครงสร้างของหน้าเว็บมีความหมายและเป็นระเบียบ คุณจะต้องแสดงรูปภาพ ชื่อสินค้า คำอธิบาย ราคา และปุ่มสำหรับเพิ่มสินค้าลงในรถเข็น การจัดวางองค์ประกอบทั้งหมดจะต้องใช้แท็ก HTML เชิงความหมายที่เหมาะสม

สิ่งที่โจทย์ต้องการ:

  • ใช้แท็ก HTML เชิงความหมาย (Semantic HTML) เช่น article, figure, img, figcaption, h2, p, button, section, div ให้เหมาะสมกับการ์ดสินค้า
  • ต้องมีการแสดงรูปภาพสินค้า โดยใช้แท็ก img พร้อมแอตทริบิวต์ alt ที่เหมาะสม
  • ต้องมีชื่อสินค้า โดยใช้แท็กหัวข้อที่เหมาะสม เช่น h2 หรือ h3
  • ต้องมีคำอธิบายสั้นๆ ของสินค้า โดยใช้แท็กย่อหน้า p
  • ต้องมีราคาของสินค้า โดยใช้แท็กที่เหมาะสม เช่น p หรือ span
  • ต้องมีปุ่ม "เพิ่มลงตะกร้า" (Add to Cart) โดยใช้แท็ก button
  • โครงสร้าง HTML ควรมีความยืดหยุ่นพอที่จะรองรับการแสดงผลบนอุปกรณ์ต่างๆ ได้ (responsive-friendly structure)
  • ห้ามใช้ CSS หรือ JavaScript ในการท้าทายนี้ ให้เน้นที่โครงสร้าง HTML เท่านั้น
