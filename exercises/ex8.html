Ex.8 : CSS : intermediate : สร้างเลย์เอาต์การ์ดสินค้าแบบ Responsive

โจทย์นี้จะให้คุณสร้างเลย์เอาต์การ์ดสินค้าที่สามารถปรับเปลี่ยนได้ตามขนาดหน้าจอ (Responsive Product Card Layout) โดยใช้ HTML และ CSS คุณจะต้องสร้างการ์ดสินค้าหลายๆ ใบที่จัดเรียงกันอย่างสวยงามบนหน้าจอขนาดใหญ่ และเรียงซ้อนกันเป็นแนวตั้งพร้อมปรับขนาดให้เหมาะสมเมื่ออยู่บนหน้าจอขนาดเล็ก โดยใช้ Media Queries และ Flexbox หรือ CSS Grid

สิ่งที่โจทย์ต้องการ:

  • ใช้ HTML และ CSS เพื่อสร้างเลย์เอาต์ของส่วนที่แสดงการ์ดสินค้าหลายใบ
  • จัดเรียงการ์ดสินค้าโดยใช้ Flexbox หรือ CSS Grid ภายในคอนเทนเนอร์หลัก
  • ใช้ Media Queries เพื่อให้เลย์เอาต์ตอบสนองต่อขนาดหน้าจอ: บนหน้าจอที่มีความกว้างตั้งแต่ 1024px ขึ้นไป: แสดงการ์ด 4 คอลัมน์ บนหน้าจอที่มีความกว้างระหว่าง 768px ถึง 1023px: แสดงการ์ด 2 คอลัมน์ บนหน้าจอที่มีความกว้างน้อยกว่า 768px: แสดงการ์ด 1 คอลัมน์เต็มความกว้าง
  • การ์ดแต่ละใบควรมี: รูปภาพ, ชื่อสินค้า, ราคา, และปุ่ม 'เพิ่มลงรถเข็น'
  • เพิ่มสไตล์พื้นฐาน เช่น ขอบมน, เงา, ระยะห่าง, และการจัดตำแหน่งข้อความให้การ์ดดูสวยงามและน่าสนใจ
