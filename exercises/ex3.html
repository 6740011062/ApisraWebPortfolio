Ex.3 : CSS : beginner : การ์ดข้อมูลส่วนตัวแบบง่าย

โจทย์นี้ คุณจะต้องสร้างคอมโพเนนต์การ์ดข้อมูลส่วนตัว (Profile Card) โดยใช้ HTML สำหรับโครงสร้าง และ CSS สำหรับการจัดรูปแบบให้สวยงามและน่าอ่าน เป้าหมายคือการฝึกใช้คุณสมบัติ CSS พื้นฐาน เช่น การจัดตำแหน่ง, สี, ขนาด, เงา และขอบโค้งมน

สิ่งที่โจทย์ต้องการ:

  • กำหนดพื้นหลังของการ์ด .profile-card ให้เป็นสีอ่อน และเพิ่ม box-shadow เพื่อให้ดูมีมิติ
  • ตั้งค่า width ของการ์ดให้ประมาณ 300px และใช้ margin: 20px auto; เพื่อจัดให้อยู่กึ่งกลางหน้าจอ
  • กำหนด padding รอบๆ ภายใน .profile-card ประมาณ 20px และเพิ่ม border-radius เล็กน้อย
  • จัดเนื้อหาทั้งหมดภายใน .profile-card ให้อยู่กึ่งกลางด้วย text-align: center;
  • ทำให้รูปภาพ .profile-img เป็นวงกลม โดยใช้ border-radius: 50%; และตั้งค่า display: block; margin: 0 auto 15px auto; เพื่อจัดกึ่งกลางและเว้นระยะห่างด้านล่าง
  • จัดรูปแบบข้อความ h2 .profile-name ให้มี font-size ที่ใหญ่ขึ้นและ margin-bottom เล็กน้อย
  • จัดรูปแบบข้อความ p .profile-title ให้มีสีเทาอ่อนลง และ font-size ที่เหมาะสม
  • ออกแบบปุ่ม .contact-button ให้มีสีพื้นหลังที่น่าสนใจ , สีข้อความเป็นสีขาว, มี padding ที่เหมาะสม, และมี border-radius` เพื่อให้ขอบมน
